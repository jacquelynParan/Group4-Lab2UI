# Group4-Lab2UI
SBJ Hardware – Employee Attendance System
📌 Project Description

This project is a simple Employee Attendance System for SBJ Hardware. It is built using HTML, CSS, and JavaScript to allow employees to clock in and clock out, and for managers to view attendance logs.

The system focuses on:

User-friendly interface
Simple and clear workflow
Responsive design for different screen sizes
Accessibility and keyboard navigation
🚀 Features
✅ Clock-in / Clock-out functionality
✅ Employee attendance logs (viewable by admin/manager)
✅ Responsive design (mobile, tablet, desktop)
✅ Keyboard navigation (Tab + Enter)
✅ Accessible interface (ARIA labels, proper contrast)

📂 Project Structure
SBJ-Attendance/
 ├── index.html         # Login / main page
 ├── dashboard.html     # Manager dashboard
 ├── style.css          # Styles
 └── README.md
 
🛠️ How to Run the Project
Option 1: Clone the Repository (Recommended)

Anyone can copy the project using Git:

git clone https://github.com/jacquelynParan/SBJ-Attendance.git

Then:

Open the folder in VSCode
Open index.html in your browser (Edge, Chrome, Firefox)
Use the interface to test clock-in and clock-out

Note: Live Server is optional. Opening index.html directly works fine.

Option 2: Open Directly
Download the folder or copy the files from GitHub
Open index.html directly in your browser
Interact with the system to test attendance functionality
🧪 Testing & Requirements
Device Simulation: Tested on mobile, tablet, and desktop using browser DevTools
Keyboard Navigation: All buttons and links accessible using Tab + Enter
Accessibility Audit: Basic ARIA labels added and color contrast checked
